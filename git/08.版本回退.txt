		//方法1:
		//根据具体的索引值也回退到具体的版本,可向前向后移动
		git reset --hard 版本的索引值
		例如:git reset --hard 02dfe2

		方法2:
		使用^符号,只能向前面的版本移动
		git reset --hard head^
		一个^表示向前移动一个版本，多个^表示向前移动多个版本
		git reset --hard head^^^
		向前移动3个版本
		
		方法3:
		使用~符号，只能向前移动若干版本
		git reset --hard head~1
		向前移动一个版本
		git reset --hard head~3
		向前移动三个版本
